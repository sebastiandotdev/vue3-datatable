# Props

All available props for the `<vue3-datatable>` component.

| Prop                    | Type                    | Default                               | Description                                                                                                                                                                                                                                                                                                                                                                                    |
| ----------------------- | :---------------------- | ------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **columns** (required)  | `array<colDef>`         | `[]`                                  | Table column definitions. See [Column Options](./columns.md).                                                                                                                                                                                                                                                                                                                                  |
| **rows** (required)     | `array<any>`            | `[]`                                  | Table row data.                                                                                                                                                                                                                                                                                                                                                                                |
| **isServerMode**        | `boolean`               | `false`                               | Set to `true` to enable server-side pagination.                                                                                                                                                                                                                                                                                                                                                |
| **totalRows**           | `number`                | `0`                                   | Total number of rows. Required when `isServerMode` is `true`.                                                                                                                                                                                                                                                                                                                                  |
| **skin**                | `string`                | `"bh-table-striped bh-table-hover"`   | Custom class for table skin. Available values: `bh-table-striped` (striped rows), `bh-table-hover` (hover highlight), `bh-table-bordered` (bordered rows), `bh-table-compact` (compact spacing).                                                                                                                                                                                               |
| **loading**             | `boolean`               | `false`                               | Show loading indicator.                                                                                                                                                                                                                                                                                                                                                                        |
| **hasCheckbox**         | `boolean`               | `false`                               | Enable row checkboxes for selection.                                                                                                                                                                                                                                                                                                                                                           |
| **search**              | `string`                | `""`                                  | Global search string applied across all searchable columns.                                                                                                                                                                                                                                                                                                                                    |
| **page**                | `number`                | `1`                                   | Current page number.                                                                                                                                                                                                                                                                                                                                                                           |
| **pageSize**            | `number`                | `10`                                  | Number of rows displayed per page.                                                                                                                                                                                                                                                                                                                                                             |
| **pageSizeOptions**     | `array<number>`         | `[10, 20, 30, 50, 100]`               | Available page size choices in the dropdown.                                                                                                                                                                                                                                                                                                                                                   |
| **showPageSize**        | `boolean`               | `true`                                | Show or hide the page size dropdown.                                                                                                                                                                                                                                                                                                                                                           |
| **rowClass**            | `array<string>` \| `function` | `""`                            | Custom class(es) applied to each row. Accepts a static array or a function `(row) => string`.                                                                                                                                                                                                                                                                                                  |
| **cellClass**           | `array<string>` \| `function` | `""`                            | Custom class(es) applied to each cell. Accepts a static array or a function `(row) => string`.                                                                                                                                                                                                                                                                                                 |
| **sortable**            | `boolean`               | `true`                                | Enable column sorting globally.                                                                                                                                                                                                                                                                                                                                                                |
| **sortColumn**          | `string`                | `"id"`                                | Name of the default sort column.                                                                                                                                                                                                                                                                                                                                                               |
| **sortDirection**       | `string`                | `"asc"`                               | Default sort direction. Accepted values: `"asc"`, `"desc"`.                                                                                                                                                                                                                                                                                                                                   |
| **columnFilter**        | `boolean`               | `false`                               | Enable per-column filter controls.                                                                                                                                                                                                                                                                                                                                                             |
| **columnFilterLang**    | `Record<string, string>` | `null`                               | Translations for column filter labels. Example: `{ no_filter: 'Aucun', contain: 'Contiens', not_contain: 'Ne contiens pas', equal: 'Egale', not_equal: 'Différent', start_with: 'Commence par', end_with: 'Termine par', greater_than: 'Supérieur à', greater_than_equal: 'Sup. ou égale à', less_than: 'Inférieur à', less_than_equal: 'Inf. ou égale à', is_null: 'Est null', is_not_null: 'Non null' }` |
| **pagination**          | `boolean`               | `true`                                | Enable pagination controls.                                                                                                                                                                                                                                                                                                                                                                    |
| **showNumbers**         | `boolean`               | `true`                                | Show page number buttons in pagination.                                                                                                                                                                                                                                                                                                                                                        |
| **showNumbersCount**    | `number`                | `5`                                   | How many page number buttons to display at once.                                                                                                                                                                                                                                                                                                                                               |
| **showFirstPage**       | `boolean`               | `true`                                | Show the "first page" button in pagination.                                                                                                                                                                                                                                                                                                                                                    |
| **showLastPage**        | `boolean`               | `true`                                | Show the "last page" button in pagination.                                                                                                                                                                                                                                                                                                                                                     |
| **firstArrow**          | `string`                | Default arrow                         | Custom HTML/icon for the first page arrow.                                                                                                                                                                                                                                                                                                                                                     |
| **lastArrow**           | `string`                | Default arrow                         | Custom HTML/icon for the last page arrow.                                                                                                                                                                                                                                                                                                                                                      |
| **previousArrow**       | `string`                | Default arrow                         | Custom HTML/icon for the previous page arrow.                                                                                                                                                                                                                                                                                                                                                  |
| **nextArrow**           | `string`                | Default arrow                         | Custom HTML/icon for the next page arrow.                                                                                                                                                                                                                                                                                                                                                      |
| **paginationInfo**      | `string`                | `"Showing {0} to {1} of {2} entries"` | Template string for the pagination info text. `{0}` = first item, `{1}` = last item, `{2}` = total.                                                                                                                                                                                                                                                                                           |
| **noDataContent**       | `string`                | `"No data available"`                 | Message shown when there are no rows to display.                                                                                                                                                                                                                                                                                                                                               |
| **stickyHeader**        | `boolean`               | `false`                               | Fix the table header while scrolling vertically.                                                                                                                                                                                                                                                                                                                                               |
| **height**              | `string`                | `"450px"`                             | Table height. Only applies when `stickyHeader` is `true`.                                                                                                                                                                                                                                                                                                                                      |
| **stickyFirstColumn**   | `boolean`               | `false`                               | Fix the first column while scrolling horizontally.                                                                                                                                                                                                                                                                                                                                             |
| **cloneHeaderInFooter** | `boolean`               | `false`                               | Repeat the header row in the table footer.                                                                                                                                                                                                                                                                                                                                                     |
| **selectRowOnClick**    | `boolean`               | `false`                               | Toggle the row's checkbox when clicking anywhere on the row.                                                                                                                                                                                                                                                                                                                                   |
